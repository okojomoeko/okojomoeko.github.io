自分用のメディアファイル名変更スクリプト。画像ファイルが作成された日時を元にリネームして、日付のフォルダを作成して整理する。とりあえず mp4 と jpg,png を振り分けたかったのでそれだけ対応。適当に書いているのでエラーハンドリングなし！！！

## 使用技術

- Shell Script
