水道光熱費を GoogleDrive 上のエクセルに登録する LINE-bot. GoogleDrive 上に配置したスクリプトを LINE-bot の WebHook にした(気がする)。これも個人的には結構便利で、シンプルなフォーマットで登録が可能. CRUD 処理はできるようにしないとなぁ…

## 使用技術

- GoogleAppScript
- GoogleCloudFunctions
- LINE Message API
