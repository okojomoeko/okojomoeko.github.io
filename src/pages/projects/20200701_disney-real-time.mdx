ディズニーランド、ディズニーシーのリアルタイム待ち時間を教えてくれる LINE-bot. LINE のトーク画面で"land"とか"sea"とか送信するとそれぞれのテーマパークのリアルタイムの待ち時間を返してくれる. GCP の無料枠で運用しているため個人用

## 使用技術

- Node.js
- Google Cloud Functions
- LINE Message API
